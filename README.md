# homebrew-tap

Custom Homebrew tap for [mariodian](https://github.com/mariodian) apps.

## Usage

```bash
brew tap mariodian/tap
```

## Casks

| Cask | Description |
|------|-------------|
| `pincer` | Desktop monitoring for local AI agents |

### Install Pincer

```bash
brew install --cask mariodian/tap/pincer
```
